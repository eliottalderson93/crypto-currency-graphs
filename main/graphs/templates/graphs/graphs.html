Displaying and translating API data into statistical graphs with Python libraries

dependencies include: Django, Pandas, Bokeh, and Requests

A custom-built RESTful Bokeh API in Python and Node.js

The request is sent via HTTP to my Python server, which modifies and sends it on to my Node.js proxy server

The proxy server sends and receives a JSON request externally, which it sends back to my Python server.

My Python server processes the received data first into the data I want and then into a Bokeh HTML component, which is displayed on the page.
